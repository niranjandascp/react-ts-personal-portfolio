# 🚀 React-TS Personal Portfolio

[![React](https://img.shields.io/badge/React-19.0-61DAFB?logo=react&logoColor=white)](https://react.dev/)

[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4.0-06B6D4?logo=tailwindcss&logoColor=white)](https://tailwindcss.com/)
[![Framer Motion](https://img.shields.io/badge/Framer_Motion-12.0-00CCAF?logo=framer&logoColor=white)](https://www.framer.com/motion/)

A high-performance, professional developer portfolio built with a modern tech stack. Designed with a focus on aesthetics, accessibility, and smooth user experience.

---

## ✨ Key Features

- **🌐 Modern UI/UX**: Clean, professional design with smooth Framer Motion animations.
- **🌓 Dark Mode**: Full support for system-preferred or manual theme switching.
- **📱 Fully Responsive**: Optimized for all screen sizes from mobile to ultra-wide desktops.
- **🚀 Performance Optimized**: Fast loading times with code splitting and lazy loading.
- **🛠️ Component-Driven**: Modular architecture using React 19 and TypeScript for type safety.
- **📊 Dynamic Integrations**:
  - **GitHub**: Real-time contribution calendar and repository stars.
  - **LeetCode**: Statistics visualization.
  - **Formspree**: Fully functional contact form.
- **🔍 SEO & Analytics**: Integrated SEO components, structured data, and Vercel Analytics.

---

## 🛠️ Tech Stack

- **Core**: React 19, TypeScript
- **Styling**: Tailwind CSS (v4), Vanilla CSS
- **Animations**: Framer Motion, Lottie React
- **Icons**: Lucide React, React Icons
- **Build Tool**: Vite
- **Integrations**: 
  - [react-github-calendar](https://github.com/grubersjoe/react-github-calendar)
  - [react-slick](https://github.com/akiran/react-slick) for carousels
  - [Formspree](https://formspree.io/) for backend-less forms
- **Deployment**: Optimized for Vercel

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/niranjandascp/react-ts-personal-portfolio.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
